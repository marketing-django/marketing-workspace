## Competitors analysis

- Direct drafting of what the strategy should cover
  - **Positioning:**
    - What is the community/Django best known for
    - What vibe does it give amongst users (e.g., devs, other products in the ecosystem)? Stability, beginner-friendly, robust community, etc.
  - **Direct competitors**: other communities/frameworks within the Python ecosystem?
    - FastAPI
  - **Indirect competitors:** other languages/full-stack
    - Laravel
    - Next.js
  - **Channel Analysis**: platforms used / content types (tutorials, updates, releases, etc.).
