# Marketing strategy draft

_**Context**_**:** We’re writing a top-down marketing strategy for Django so we can better use our limited contributor marketing resources on the right problems. We’re doing it as a rough first draft so we unblock [website redesign](https://github.com/django/djangoproject.com/issues/2287) work as soon as possible.

## Introduction & context

### 2026 framing

_200 words, Q: “why do we need this now?”_

1. _Marketing as a historical gap in Django_
   1. _It’s happening ad-hoc, it’s important, we lack the expertise_
2. Website redesign \- the website is our \#1 marketing asset
3. Sustain the work of existing volunteers \- specifically Fundraising WG
4. Keeping Django compelling for devs in an age of uncertainty (AI, new frameworks, shifting expectations)
5. Demonstrate the value for organizations \- Django is Enterprise™ Ready™
6. [Debunking Django myths](https://www.youtube.com/watch?v=yyX0QoUzoE4)
   1. Popularity of Django with learners vs. established devs
