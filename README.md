# [Django marketing workspace](https://marketing-django.github.io/marketing-workspace/)

A shared workspace to support our work on a new marketing strategy for Django and follow-on marketing activities.

## Contents

We organize information across four layers:

1. [Contributing](./docs/contributing/): guidance on how to work within this workspace. Read this first, for all types of tasks.
2. [Knowledge base](./knowledge-base/): general-purpose information about Django, relevant for a wide range of projects.
3. [Marketing strategy](./marketing-strategy/): information and working documents related to marketing specifically, across a wide range of projects.
4. [Website redesign](./website-redesign/): information about the website redesign project specifically.
