# [Django marketing workspace](https://marketing-django.github.io/marketing-workspace/)

A shared workspace to support our work on a new marketing strategy for Django and follow-on marketing activities.

## Contents

We organize information across four layers:

1. [Contributing](./docs/contributing/README.md): guidance on how to work within this workspace. Read this first, for all types of tasks.
2. [Knowledge base](./docs/knowledge-base/README.md): general-purpose information about Django, relevant for a wide range of projects.
3. [Marketing strategy](./docs/marketing-strategy/README.md): information and working documents related to marketing specifically, across a wide range of projects.
4. Projects, like [Website redesign](./docs/website-redesign/README.md): information about the website redesign project specifically.
